Working with github continuation

we learnt to make repositories and connecting the remote repository to the local repository. To do this you;
1.A new terminal in your code editor
2.Type cd space the name of the folder you're working with
3.Go to github, create a remote repository for the folder
4.From github copy and paste the first line in the listed links
5.Paste this link into the code editorand click enter
6.In descending order copy and paste from github into the code editor
7.After the last link is pasted and entered, type git space add space "." then enter. This adds all files within the folder
8.Next type git space commit space -m space "your comment"
9.Finally type git space push. This ensures that everything is linked.
